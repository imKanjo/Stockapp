# Система управления складом

Современное веб-приложение для управления складскими операциями, включая перемещение товаров, отгрузку и учет инвентаря.

## Функционал

### Роли пользователей
- **Администратор**: полный доступ к системе, управление пользователями, зонами, ячейками и категориями
- **Менеджер**: доступ к отчетам, статистике, истории операций, карте склада
- **Работник**: выполнение операций с товарами (приемка, перемещение, отгрузка)

### Основные возможности
- Приемка товаров в ячейки склада
- Перемещение товаров между ячейками
- Отгрузка товаров со склада
- Просмотр истории операций
- Учет инвентаря
- Управление зонами и ячейками
- Генерация отчетов

## Требования

- Node.js (версия 14 или выше)
- npm или yarn

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

или если используете yarn:
```bash
yarn install
```

2. Запустите сервер:
```bash
node backend/server.js
```

3. Приложение автоматически откроется в браузере по адресу `http://localhost:4000/login.html`

## Структура приложения

- `frontend/` - клиентская часть (HTML, CSS, JavaScript)
- `backend/` - серверная часть (Node.js + Express)
- `database/` - папка для хранения базы данных (SQLite)

## API endpoints

- `POST /api/users/login` - аутентификация пользователя
- `GET /api/products` - получение списка товаров
- `GET /api/cells` - получение списка ячеек
- `POST /api/inventory` - добавление товара в ячейку
- `POST /api/move/simple` - перемещение товара
- `POST /api/take/simple` - отгрузка товара
- `GET /api/operations` - история операций
- `GET /api/reports/inventory` - отчет по остаткам

## Особенности реализации

- Используется SQLite для хранения данных
- Аутентификация пользователей по идентификатору
- Реализована ролевая модель доступа
- Автоматическое открытие браузера при запуске сервера
- Интернационализация на русском языке
- Адаптивный интерфейс для разных ролей пользователей
